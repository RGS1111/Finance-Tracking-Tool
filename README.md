# Finance-Tracking-Tool
Finance Tracking Tool to input and store spendings and simplify finance. Initial CLI tool created to test basic programming skills.
.env file was created by creating a new .env file and then storing the key within it.
To access a .env file, the 'dotenv' module must be installed. : https://www.geeksforgeeks.org/python/how-to-create-and-use-env-files-in-python/
The 'dotenv' module is used to load up the .env file within the same folder where load_dotenv() is run.
